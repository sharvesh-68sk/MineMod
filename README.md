# MineMod

MineMod is a static web app that lets players choose Minecraft mods with checkboxes and download a ready-to-import `.mrpack` file in one click.

## Features

* Dark gaming-themed UI
* Checkbox-based mod selection
* One-click `.mrpack` download generated entirely in browser JavaScript
* No backend required (works on GitHub Pages)

## Run locally

Open `index.html` in any modern browser.
